Sales Dashboard Analysis (Excel)

A sales analysis dashboard created using Microsoft Excel to evaluate business performance and transform transaction data into easy-to-understand insights.

Objectives

• Analyze sales trends over time
• Identify products with the highest revenue contribution
• Understand customer purchasing behavior
• Evaluate customer satisfaction

Dataset

• Customer Name
• Product Name
• Quantity
• Total Sales
• Rating
• Sales Date (Monthly)

Key Metrics

• Total Sales
• Average Quantity per Transaction
• Sales by Month
• Top Products by Revenue
• Customer Contribution

Insights

The dashboard highlights customer purchasing patterns and product performance. Some products contribute more significantly to revenue, and there are variations in purchasing behavior across different periods. These insights can be used to inform sales strategies and inventory planning.

Tools

• Microsoft Excel
• Pivot Tables
• Data Cleaning
• Dashboard Design
