Poverty Indicators Analysis ‚Äî Tableau Story

Project ini merupakan analisis data indikator kemiskinan yang divisualisasikan menggunakan Tableau dalam format data storytelling. Visualisasi dirancang untuk memberikan gambaran komprehensif mengenai kondisi kemiskinan melalui beberapa indikator utama dan perbandingan tren.

Objectives

  • Memahami kondisi kemiskinan berdasarkan indikator statistik
  • Mengidentifikasi tren perubahan kemiskinan dari waktu ke waktu
  • Menyajikan insight secara naratif agar mudah dipahami non-teknis

Indicators Analyzed

Analisis berfokus pada indikator kemiskinan yang umum digunakan dalam statistik sosial:

  • Persentase Penduduk Miskin (P0)
  • Kedalaman Kemiskinan (P1)
  • Keparahan Kemiskinan (P2)

Indikator ini digunakan untuk mengukur tidak hanya jumlah penduduk miskin, tetapi juga tingkat kerentanan serta kesenjangan kondisi ekonomi.

Key Insights

  • Visualisasi menunjukkan dinamika tingkat kemiskinan antar periode
  • Perubahan indikator memberikan gambaran tingkat kesenjangan ekonomi
  • Storyboard membantu menjelaskan konteks data secara berurutan

Tools

  • Tableau Public
  • Data Visualization
  • Data Storytelling

Outcome

Project ini menunjukkan kemampuan dalam mengolah data sosial menjadi insight, membuat visualisasi yang komunikatif, serta menyusun narasi berbasis data untuk mendukung pengambilan keputusan.
