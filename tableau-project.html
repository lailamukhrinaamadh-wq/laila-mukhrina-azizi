Poverty Indicators Analysis — Tableau Story

This project is an analysis of poverty indicator data visualized using Tableau in a data storytelling format. The visualizations are designed to provide a comprehensive overview of poverty conditions through several key indicators and trend comparisons.

Objectives

  • Understand poverty conditions based on statistical indicators
  • Identify trends in poverty changes over time
  • Present insights in a narrative format for easier understanding by non-technical audiences

Indicators Analyzed

The analysis focuses on commonly used poverty indicators in social statistics:

  • Poverty Rate (P0)
  • Poverty Gap Index (P1)
  • Poverty Severity Index (P2)

These indicators are used to measure not only the proportion of people living in poverty, but also the level of vulnerability and economic inequality.

Key Insights

  • The visualizations reveal the dynamics of poverty levels across periods
  • Changes in indicators provide insight into economic inequality
  • The storyboard helps explain the data context in a sequential manner

Tools

  • Tableau Public
  • Data Visualization
  • Data Storytelling

Outcome

This project demonstrates the ability to transform social data into insights, create clear and communicative visualizations, and develop data-driven narratives to support decision-making.
